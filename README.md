**========== decimal to binary conversion ===============**

 suppos number is 52
 52/2 = 26 => 0
 26/2 = 13 => 0
 13/2 = 6 => 1
 6/2 = 3 => 0
 3/2 = 1 => 1
 3/2 = 1 
 binary is = 110100

 suppos number is 27
 27/2 = 13 => 1
 13/2 = 6 => 1
 6/2 = 3 => 0
 3/2 = 1 => 1
 3/2 = 1 
 binary is = 11011

**========== binary to decimal conversion =============**

11011 => 1*2p4 + 1*2p3 + 0*2p2 + 1*2p1 + 1*2p0
      => 1*16  + 1*8   + 0*4   + 1*2   + 1*1
      => 16    + 8     + 0     + 2     +1
      => 27
